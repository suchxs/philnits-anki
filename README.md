# philnits-anki
Philnits Exam Anki Reviewer Deck

## Installation
- Install [Anki](https://apps.ankiweb.net/)
- File -> Import -> PhilNITS.apkg
- Decide how many cards you want to study daily (Deck options -> Daily Limits -> New Cards / day)

## Why Anki?
- [Anki](https://apps.ankiweb.net/) is better for long-term retention because it uses a dedicated spaced-repetition engine (Currently FSRS is god-tier) designed to optimize when you review each card, while Obsidian flashcards are generally simpler and less adaptive. With Anki’s FSRS scheduler, intervals are personalized based on your actual recall performance, so you study more efficiently and waste less time on cards you already know. In practice, this means faster progress and better memory retention with fewer total reviews. This is incredibly helpful for the PhilNITS exam review if you're like me who procrastinates alot

## Converter
The converter uses:

- Front: content between the note title and `?`
- Back: content between `?` and `---`
- Tags: `Category`, `Status`, year folder, and note ID
